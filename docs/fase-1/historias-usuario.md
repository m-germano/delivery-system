# Histórias de Usuário

## Cliente

- Como cliente, eu quero criar minha conta para poder fazer pedidos.
- Como cliente, eu quero visualizar produtos disponíveis para escolher o que comprar.
- Como cliente, eu quero criar um pedido para receber os produtos no meu endereço.
- Como cliente, eu quero acompanhar o status do pedido para saber quando ele será entregue.

## Empresa

- Como empresa, eu quero cadastrar produtos para disponibilizá-los aos clientes.
- Como empresa, eu quero editar e desativar produtos para manter o cardápio atualizado.
- Como empresa, eu quero visualizar pedidos recebidos para preparar as entregas.
- Como empresa, eu quero atualizar o status do pedido para informar o andamento ao cliente.

## Entregador

- Como entregador, eu quero ficar disponível para receber entregas.
- Como entregador, eu quero visualizar pedidos disponíveis para aceitar uma corrida.
- Como entregador, eu quero atualizar o status da entrega para manter o cliente informado.
